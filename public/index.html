Music Autocomplete

Usage

API URL

https://musicautocomplete.deno.dev/search?q=your+query

Example responses

Response for query "queen" (formatted):

{
    "error": false,
    "results": [
        "Bohemian Rhapsody",
        "Love Of My Life",
        "I Want To Break Free",
        "The Show Must Go On",
        "Too Much Love Will Kill You",
        "I Want To Break Free (Single Remix)",
        "I Want It All",
        "Don't Stop Me Now (...Revisited)",
        "Bohemian Rhapsody (Live Aid)",
        "Good Old-Fashioned Lover Boy"
    ]
}

Response for query with no result:

{
    "error": false,
    "results": []
}
All responses are JSON encoded.

Response Types

type ApiResponse = {
    error: true;
    type: 'no-query' | 'invalid' | 'server-error';
    results: [];
} | {
    error: false;
    results: string[];
};
You'll need to handle errors and results in your code to make sure there is no unexpected behavior.
